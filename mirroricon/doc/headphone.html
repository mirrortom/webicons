耳机

头戴式声音

<i class="i-a-headphone"></i>

头戴式耳机分为圆圈和耳机喇叭两部分.圆圈部分:before为0.6em正方形,上边圆角,下边无.水平居中.

喇叭是:after的两个阴影,宽高比0.2em:0.3em,椭圆形.位于左边和下边0.1em处,右边喇叭位置对称.


完整样式

.i-a-headphone
    &:before,&:after
        left: 50%
        transform: translateX(-50%)

    &:before
        top: 10%
        width: .6em
        height: .6em
        border: @css{max(.04em,1px)} solid
        border-radius: 50% 50% 0 0
        border-bottom: none

    &:after
        top: 60%
        width: .2em
        height: .3em
        border-radius: 50%
        box-shadow: .3em 0,-.3em 0
