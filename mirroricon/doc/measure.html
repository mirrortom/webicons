尺

测量

<i class="i-a-measure"></i>

尺身是:before,长度1em与容器等长,宽度0.3em,旋转45度.水平垂直居中.

刻度是:after和它的阴影,宽度1px,高度为尺高度一半:0.15em.也居中.

刻度要对齐在尺的上边线.只设置居中时如下.


刻度离尺边有一定距离,由于尺和刻度都是居中,所以,刻度在尺中也是居中的,那么刻度离上边的距离就是 (尺-刻度)/2 = (.3em - .15em) / 2 = 0.075em

css变换的效果是叠加的,所以,先是居中,再旋转,然后移动Y坐标0.075em即可.

transform: translate(-50%,-50%) rotate(-45deg) translateY(-.075em)


完整样式

.i-a-measure
    &:before, &:after
        left:50%
        top:50%

    &:before
        transform: translate(-50%,-50%) rotate(-45deg)
        width: 1em
        height: .3em
        border: 1px solid

    &:after
        transform: translate(-50%,-50%) rotate(-45deg) translateY(-.075em)
        width: 1px
        height: .15em
        background-color: currentColor
        box-shadow: .16em 0,.32em 0,-.16em 0,-.32em 0
