菜单

3条横杠的菜单图标.八卦中的乾卦也是它.3个点的菜单图标,是一种变形.

<i class="i-f-menu"></i>

<i class="i-f-menu dot"></i>

菜单图标就是在减号图标的基础上加了两个阴影,就变成3条横杠了.减号(中间横杠)的长度还是1em,厚度也是.1em.

上边和下边的横杠是减号的阴影,阴影位置应该是图标的顶部和底部.但是希望阴影离图标顶底留出一点距离,借鉴了参考项目后,设置阴影离减号的距离是1.5个减号厚度,即.15em.这样3条横杠,2条间距,总厚度是,.1em*3 + .15em*2 = .6em.总高度未超过图标高度(1em).

width: 100px;
  height: 10px;
  // 上,下横杠(阴影)位置,离减号(中间横杠)1.5个减号厚度的距离
  box-shadow: 0 25px #00000060,0 -25px #00000060;

3个点的形式就是改变了横杠的长度,将长度和宽度改成一样的就实现了,其它不变.

.dot
  width: 10px;
  height: 10px;

完整样式

.i-f-menu
    &:before
        width: 1em
        height: @css{max(.1em,2px)}
        background-color:currentColor
        box-shadow: 0 .25em,0 -.25em
    // 3个点菜单,就是改width就可
    &.dot:before
        width: @css{max(.1em,2px)}
