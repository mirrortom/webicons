声音

声音,表示音量

<i class="i-a-sound"></i>

图标分为喇叭和声波两部分

喇叭使用:before,是一个竖立梯形,左右两个底为1:2,右边底为0.6em,宽度.2em

声波是:after,是一个边长为.6em的正方形旋转45度后,以左下角顶点为渐变中心的径向渐变,实现3条波纹.声波的实现参考wlan图标

垂直居中,水平位置经过调整设置为,喇叭left:10%,声波盒模型right:0.


完整样式

.i-a-sound
    &:before,&:after
        top:50%

    &:before
        left:10%
        transform:translateY(-50%)
        width:.2em
        height:.6em
        border-color: transparent
        border-style: solid
        border-width: .15em .2em .15em 0
        border-right-color: currentColor

    &:after
        right: 0
        width: .6em
        height: .6em
        transform: translateY(-50%) rotate(45deg)
        background-image: radial-gradient(circle .6em at left bottom,
                          transparent 30%,currentColor 32% calc(32% + max(.04em,1px)),
                          transparent calc(34% + max(.04em,1px)) 60%,currentColor 62% calc(62% + max(.04em,1px)),
                          transparent calc(64% + max(.04em,1px)) 90%,currentColor 92% calc(92% + max(.04em,1px)),
                          transparent)
