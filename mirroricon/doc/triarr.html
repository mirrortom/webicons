三角箭头

三角形箭头图标,是等边三角形,箭头分上下左右.

<i class="i-f-triarr"></i>
  <i class="i-f-triarr down"></i>
  <i class="i-f-triarr left"></i>
  <i class="i-f-triarr right txt-info"></i>

自然是边框法做三角形,要求是等边三角形,需要计算边框长度.假设边长是 L,先做一个底边水平,底边对应顶角朝下的三角形.

等边三角形的高h,与边长L的关系 h = sin60 * L

根据边框法做三角形规律,要做出这个等边三角形,border-left和border-right的厚度是 L/2,而border-top的厚度是 h.下边框则不需要了,none.

width: 0;
  height: 0;
  border: 50px solid transparent;
  // 上边框厚度就是三角形的高,下边框去掉
  border-top: calc(0.866 * 100px) solid;
  border-bottom: none;

方向向下(bottom)的三角形箭头,需要的是top边框,并且要去掉bottom边框,这是个规律.据此画出其它3个方向的.

但是,只画一个方向的,其它3个方向的旋转90度得到,更加方便.

由于边长是 L,高是 sin60 * L,所以三角形所在的盒模型是一个长方形.需要对齐位置.

最后,三角形使用:before,图标容器使用flex布局,居中对齐.免去居中计算位置的麻烦.


完整样式

.i-f-triarr
    &:before
        width: 0
        height: 0
        border: .5em solid transparent
        border-bottom: .866em solid currentColor
        border-top: none

    &.down:before
        transform: rotate(180deg)

    &.left:before
        transform: rotate(-90deg)

    &.right:before
        transform: rotate(90deg)
