温度计

温度计,测量仪

<i class="i-a-thermometer"></i>

观察了一些温度计,一般有3部分,水银球,柱子,刻度.图标也显示这3个部分.

柱子和球水平居中,柱子离上边10%,球离下边5%.柱子下方和球上方有一点重叠.

柱子高度0.6em,宽度0.1em(最小3px),使用:before.左上/右上角圆角.柱子底部起有水银位,这个其实是border-bottom实现的.

球半径0.15em,用容器元素的径向渐变画出,圆心位置在(50%,80%)处,球体半径0.15em,这样球下方离底部还有5%的距离.

刻度线是:after的box-shadow实现,宽度0.1em(最小3px),围绕在柱子两侧,阴影离柱子的水平距离是0.05em(刻度线宽的1/2),阴影竖直间隔0.1em.左右两边刻度在垂直位置上错开0.05em,表示两种单位的刻度.


完整样式

.i-a-thermometer
background-image: radial-gradient(circle .15em at 50% 80%,currentColor 0 96%,transparent)

&:before,&:after
    left: 50%
    width: @css{max(.1em,3px)}
    transform: translateX(-50%)

&:before
    top: 10%
    height: .6em
    border: 1px solid
    border-radius: .1em .1em 0 0
    border-bottom: .25em solid

&:after
    top: 20%
    height: 1px
    box-shadow: .15em 0, .15em .1em,.15em .2em,.15em .3em,
                -.15em .05em,-.15em .15em, -.15em .25em,-.15em .35em
