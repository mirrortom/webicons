加号

<i class="i-ac-plus"></i>

这个图标比较简单.加号由横线和竖线组成,分别使用:before和:after伪元素.i标记相对定位,伪元素绝对定位.令其垂直水平居中.

其中的竖线是由横线旋转90度所得.加号的横线和竖线厚度固定设置为10%,最小2px.

display: inline-block;
width: 100px;
height: 100px;

// 横线
:before,:after
width:100px;
height:10px;

// 竖线
:after
transform: rotate(90deg)

完整样式

.i-ac-plus
    &:before, &:after
        background-color: currentColor
        width: 1em
        height: @css{max(.1em,2px)}

    &[class^=i-]:after
        transform: translate(-50%,-50%) rotate(90deg)
