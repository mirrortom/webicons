音乐

音符图标,表示音乐

<i class="i-a-music"></i>

方框是:before,上边框加粗了.底边去掉,左右为1px.长宽为: .5em : .7em.水平居中但偏右5%,离上边0.1em

音符蝌蚪形是:after,长宽比例为2:1的椭圆.box-shadow实现.


完整样式

.i-a-music
    &:before
        left: 55%
        top: .1em
        width: .5em
        height: .7em
        transform: translateX(-50%)
        border: 1px solid
        border-top: .1em solid
        border-bottom: none

    &:after
        top: 0
        left: 0
        width: @css{max(.2em,4px)}
        height: @css{max(.1em,3px)}
        border-radius: 50%
        box-shadow: .12em .75em,.6em .75em
